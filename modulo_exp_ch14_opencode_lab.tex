\section{Passo 2: Primeiro Experimento — Mercado com 2 Ativos}

\subsection{2.1 A Pergunta Cientifica (SPEC)}

\begin{center}
\fbox{\begin{minipage}{0.88\textwidth}
\textbf{SPEC-GAT-001:} Em um mercado com 2 ativos cujos precos seguem
movimento Browniano geometrico com retornos esperados $\mu_1, \mu_2$ e
taxa livre de risco $r$, a curvatura $R$ do fibrado do mercado e zero
\textbf{se, e somente se}, $\mu_1 = \mu_2 = r$?
\end{minipage}}
\end{center}

\textbf{Explicacao para leigos:} Se duas acoes tem o mesmo retorno esperado
que a poupanca, nao ha como ganhar dinheiro sem risco comprando uma e
vendendo a outra. A GAT diz que isso e equivalente a ``curvatura zero'' —
o mercado e ``plano''. Queremos verificar se isso e verdade.

\subsection{2.2 Escrever o Teste Primeiro (TDD — RED)}

\begin{lstlisting}[caption={test\_gat\_curvatura.py — Teste antes do codigo}, label=lst:tdd1]
"""SPEC-GAT-001: Test-Driven Development para curvatura de mercado"""
import numpy as np
import pytest

def simular_gbm(S0, mu, sigma, T=1.0, N=252, seed=42):
    """Simula movimento Browniano geometrico (Euler-Maruyama)"""
    np.random.seed(seed)
    dt = T / N
    S = np.zeros((N+1, len(S0)))
    S[0] = S0
    for i in range(N):
        dW = np.random.normal(0, np.sqrt(dt), len(S0))
        S[i+1] = S[i] * (1 + mu*dt + sigma*dW)
    return S

def calcular_curvatura_aproximada(S, r, dt):
    """Calcula aproximacao numerica da curvatura R"""
    N, M = S.shape[0]-1, S.shape[1]
    R = np.zeros((N, M))
    for t in range(N):
        for j in range(M):
            ret = (S[t+1, j] - S[t, j]) / max(S[t, j], 1e-8)
            R[t, j] = (ret/dt - r) / S[t, j]
    return np.mean(np.abs(R))

# TESTE (deve falhar inicialmente — RED)
def test_curvatura_zero_quando_mu_igual_r():
    """CT-1: Se mu = r para todos os ativos, curvatura ~ 0"""
    mu = np.array([0.05, 0.05])   # ambos iguais a r
    r = 0.05
    S = simular_gbm(S0=[100.0, 100.0], mu=mu, sigma=[0.2, 0.2])
    R = calcular_curvatura_aproximada(S, r, dt=1/252)
   assert R < 0.1, f"Curvatura deveria ser ~0, obtida {R:.4f}"

def test_curvatura_nao_zero_quando_mu_diferente_r():
    """CT-2: Se mu != r para algum ativo, curvatura > 0"""
    mu = np.array([0.05, 0.10])   # ativo 2 tem retorno maior
    r = 0.05
    S = simular_gbm(S0=[100.0, 100.0], mu=mu, sigma=[0.2, 0.2])
    R = calcular_curvatura_aproximada(S, r, dt=1/252)
   assert R > 0.01, f"Curvatura deveria ser >0, obtida {R:.4f}"
\end{lstlisting}

\subsection{2.4 Interpretar os Resultados (GREEN)}

Se ambos os testes passarem, verificamos que:

\begin{enumerate}[nosep]
 \item Quando $\mu_1 = \mu_2 = r$ (todos os retornos esperados igual a taxa
 livre de risco), a curvatura $R$ e aproximadamente zero — o mercado e
 ``plano'' e nao ha arbitragem.
 \item Quando $\mu_2 > r$ mas $\mu_1 = r$, a curvatura $R$ e maior que zero
 — existe uma ``curvatura'' no mercado, indicando oportunidade de
 lucro.
\end{enumerate}

\textbf{Limitacao importante (transparencia):} O teste CT-2 mostra $R > 0$
quando $\mu \neq r$, mas isso \textbf{nao} prova que existe arbitragem no
sentido estrito (risco zero). Prova apenas que a medida fisica $P$ nao e
uma medida martingal — o que e diferente de afirmar que \textbf{nenhuma}
medida equivalente $P^*$ existe. Esta distincao e exatamente o que a
condicao de Novikov captura.

\section{Limitacoes e Transparencia}

\begin{enumerate}[nosep]
 \item \textbf{Aproximacao numerica:} O calculo de $R$ neste capitulo e
 uma aproximacao de primeira ordem (diferencas finitas). A curvatura
 exata requer derivacao simbolica.
 \item \textbf{Mercado simulado:} Os precos sao gerados por GBM — um modelo
 que nao captura saltos, agrupamento de volatilidade ou
 microestrutura de mercado.
 \item \textbf{Auto-avaliacao:} Os scores do Cora-Debate sao auto-reportados.
 Nao substituem revisao por pares humana.
 \item \textbf{Custo computacional:} Para $N > 100$ ativos, o calculo de
 curvatura torna-se computacionalmente inviavel em hardware domestico.
 \item \textbf{Reprodutibilidade:} A seed (42) e fixa para garantir
 reproducao. Alterar a seed produzira resultados numericos diferentes
 mas qualitativamente equivalentes.
\end{enumerate}
