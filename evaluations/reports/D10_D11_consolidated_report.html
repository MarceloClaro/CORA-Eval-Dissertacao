📝 Lições Aprendidas

  1. IIFE captura i: (lambda i: lambda *args: ...)(i) resolve colisão entre i de loop e *dep_vals de execute_dag
  2. *args ≠ **kw: execute_dag sempre passa valores como args posicionais, nunca keyword — lambdas devem usar *args ou parâmetros nomeados explícitos
  3. Dependências = parâmetros: todo parâmetro da lambda precisa estar na lista de dependências; omissão causa TypeError: missing required positional argument
  4. Keys importam: nomes de chaves nos dicts dep e out devem corresponder exatamente — Python aceita "yield" como chave mesmo sendo keyword
  5. Não duplicar tarefas: C-N4-3 tinha versão quebrada (sem out) e redo na mesma lista — remover a quebrada resolveu
